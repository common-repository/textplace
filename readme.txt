== Description ==

Textplace is a plugin to include commonly used text across multiple posts, pages and templates. It makes maintaining repetitive text easy. Text can include HTML, so "text" can really mean links, images, anything.

== Installation ==

1. Download and unzip Textplace archive.
2. Move folder to wp-content>plugins
3. Activate plugin and navigate to Manage>Textplace
